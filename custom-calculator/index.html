How to Use Custom Calculator

Creating Conversions:

  • • Give your conversion a descriptive name
  • • Add units with conversion factors (relative to a base unit)
  • • Conversion factor = how many base units this unit represents
  • • Example: If meter is base (factor 1), then km factor = 1000

Custom Formulas:

  • • Use 'value' for the input number
  • • Use 'from' for source unit factor
  • • Use 'to' for target unit factor
  • • Example: Temperature conversion with offsets
